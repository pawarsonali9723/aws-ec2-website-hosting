# AWS EC2 Website Hosting Project 🚀

## 📌 Overview
This project demonstrates how to host a static website on an AWS EC2 instance using Apache web server.

## ⚙️ Steps Performed
- Launched EC2 instance (Amazon Linux)
- Configured security groups (SSH & HTTP)
- Installed Apache (httpd)
- Created and deployed a static HTML webpage
- Accessed the website via public IP

## 🛠️ Technologies Used
- AWS EC2
- Linux
- Apache (httpd)
- GitHub

## 🌐 Output
Website successfully hosted and accessible through public IP address.
